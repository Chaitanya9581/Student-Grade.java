1️⃣ Project Overview
The Student Grade Calculation System is a console-based Java application designed to calculate a student’s total marks, percentage, and grade based on marks entered for five subjects.
This project demonstrates the implementation of:
Basic Java syntax
User input handling
Arithmetic operations
Conditional statements
Program execution flow
It is ideal for beginners learning Java programming and fundamental problem-solving techniques.
2️⃣ Problem Statement
Develop a Java program that:
Accepts marks for five subjects.
Calculates:
Total Marks
Percentage
Assigns a grade based on predefined grading criteria.
Displays the result to the user.
3️⃣ Objectives
The primary objectives of this project are:
To understand Java program structure.
To learn how to use the Scanner class for user input.
To apply arithmetic calculations.
To implement decision-making using if-else statements.
To design a simple grading system.
